tag baru html

header = untuk membuat header pada halaman html.
main = untuk membuat bagian utama/ isi dari halaman html.
aside = membuat side bar untuk info tambahan dari halaman web
footer = membuat bagian footer pada halaman web
article = membuat halaman dengan header main footer seperti pada body
section = untuk pengelompokan tag
figure = untuk membuat gambar menjadi figure
captionfigure = membuat caption pada figure/gambar
details = membuat tulisan dengan border dan padding otomatis
summary = membuat tulisan tersembunnyi dengan anak panah untuk membuka/melihat info seperti dibawah
